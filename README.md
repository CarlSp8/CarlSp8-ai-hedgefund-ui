# AI Hedge Fund Dashboard (UI)

A production-ready UX/UI scaffold for an **AI hedge fund** web dashboard: Overview, Portfolio, Risk, Execution, Research, Investor Portal.

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts (charts)
- Radix UI (accessible primitives)

## Quickstart
```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Pages
- /overview
- /portfolio
- /risk
- /execution
- /research
- /investor

## Notes
This repo ships with mock data + reusable UI components (KPI tiles, Decision Cards, tables). Replace mock data with your backend/API.
